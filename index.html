Chapter: 3

Chapter Summary

The system unit is the case containing the electronic components that process data. Inside, the motherboard (the main circuit board) holds the processor (CPU), memory, and adapter cards. The CPU, which interprets and carries out instructions, consists of two main parts: the Control Unit (which directs operations) and the Arithmetic Logic Unit (ALU) (which performs calculations and comparisons). The processor's speed is managed by a system clock and measured in gigahertz (GHz). Computers represent data using the binary system (digits 0 and 1). A single digit is a bit, and a group of 8 bits, which can represent a single character, is called a byte.

Chapter: 4

Chapter Summary

Storage refers to hardware that records and retrieves items to and from storage media. Writing transfers items from memory to storage, while Reading transfers them from storage into memory. Common storage types include Hard Disks (which use magnetic platters) and Flash Memory, a solid-state media with no moving parts (e.g., SSDs, USB drives). Cloud Storage is an internet service for remote storage, offering universal access and backups. Optical Discs (like CDs, DVDs, and Blu-ray) are read by a laser. Data is organized in a hierarchy: a Character (one byte), a Field (related characters), a Record (related fields), and a Data File (related records).
